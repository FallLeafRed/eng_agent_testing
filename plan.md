   ```python
   import streamlit as st
   from game import simple_calculator

   def main():
       st.title('Simple Calculator')
       num1 = st.number_input("Input first number")
       num2 = st.number_input("Input second number")
       operation = st.selectbox("Select operation", ["Addition", "Subtraction", "Multiplication", "Division"])
       
       if st.button("Calculate"):
           result = simple_calculator(num1, num2, operation)
           st.write(f"Result: {result}")

   if __name__ == "__main__":
       main()
   ```

2. **Create game.py**: This file will contain the logic for the calculator operations. It should have a function that takes two inputs and an operation, performs the calculation, and returns the result:
   ```python
   def simple_calculator(num1, num2, operation):
       if operation == "Addition":
           return num1 + num2
       elif operation == "Subtraction":
           return num1 - num2
       elif operation == "Multiplication":
           return num1 * num2
       elif operation == "Division":
           if num2 != 0:
               return num1 / num2
           else:
               return "Error: Division by zero"
   ```
